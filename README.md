# Accounting Ledger App

This is a console-based Java application that allows users to manage financial transactions.

## Features
- Add deposits
- Add payments
- View all transactions
- Filter deposits and payments
- Reports (Month, Previous Month, Year, Previous Year)
- Search by vendor
- Custom search

## How to Run
1. Open the project in IntelliJ
2. Run the Main class
3. Use the menu options in the console

## Technologies Used
- Java
- File I/O (BufferedReader)
- LocalDate
